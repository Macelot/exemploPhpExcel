Este projeto tem dois diretórios, um com exemplos de PHP e outro com o vendor, para quem naõ conseguir isntalar o PHPSpredSheet com o composer.

Exemplo geraexcel = Exemplo de escrita de Planilha xls em PHP
Exemplo = leexcel Exemplo de leitura de Planilha xls em PHP

O projeto também tem o USBwebServer com PHP 7.1 para executar os exemplos.
